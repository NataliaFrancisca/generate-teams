# R&C - *Registre seus jogadores e crie times*

Projeto de uma aplicação onde você registra seus jogadores, e depois
consegue sortear times.

## Stack utilizada

**Front-end:** 
- React;
- Redux;
- React-Router;
- Styled-Components.

## Funcionalidades

- Registrar Jogadores
- Editar Jogadores
- Deletar Jogadores
- Criar Times

* *os dados de jogadores e times ficam salvos no `localStorage`*
